## NUI

不洗碗工作室的UI框架 
由雷鹏飞研发



喜欢请star，蟹蟹

## 📦 Install

```bash
npm install NUI --save
```

## 🔨 Usage

```jsx
import { Button } from 'NUI';
ReactDOM.render(<Button />, mountNode);
```

## ⌨️ Development

```bash
$ git clone https://github.com/YYL1999/NUI.git
$ cd NUI
$ npm install
$ npm run start
```
